What is DOM ? What is the purpose of DOM ?

The word DOM means Document Object Model . It is a programming interface for web documents.

With the help of this we can change the style, structure etc. of the html documents. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How to get element from DOM ?

There is many way to get element from DOM . Some of these are:
To get specific element : getElementById()
To get same class name element: getElementsByClassName()
To ger same Tag name element : getElementsByTagName() etc.

What is event bubbles ?

Event Bubbling is a concept in the DOM . It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

What is callback function ? What is the purpose of callback function ?

A callback is a function passed as an argument to another function. This technique allows a function to call another function. A callback function can run after another function has finished.

Callback is to execute code in response to an event. These events can be like mouse clicks; with callback, we can add text at the end of each statement like"execute this code every time the user clicks a key on the keyboard."
